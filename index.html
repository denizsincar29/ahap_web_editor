Keyboard shortcuts

Press H anywhere in the item list for a compact popup version of this list.

Cursor and selection

  • Left / Right - move cursor by one step, clears any selection
  • Shift+Left / Shift+Right - extend or shrink a time-range selection
  • Ctrl+Left / Ctrl+Right - move cursor by one full bar
  • Ctrl+Shift+Left / Ctrl+Shift+Right - extend the selection by a bar
  • Up / Down - switch the active track
  • Shift+Up / Shift+Down - shrink / grow the selection across tracks (start a selection, then Shift+Down to add the next track to it)
  • [ / ] - decrease / increase time step size (seconds mode)
  • Home / End - jump cursor to start / end of pattern
  • Escape - clear the selection, or stop playback if there is none
  • N - switch to Normal insert mode from anywhere (same key as MuseScore's note-input toggle)

Bars and beats

  • - (dash) - popup: set time signature and tempo, switches the cursor to bar/beat steps
  • Shift+1-Shift+9 - while in bar/beat mode, sets zoom: 1 bar, half bar, beat, half beat, triplet, quarter beat, sixth, eighth, sixteenth

Note/rest duration (MuseScore layout)

  • 1 64th, 2 32nd, 3 16th, 4 8th, 5 quarter, 6 half, 7 whole, 8 double whole (breve), 9 longa
  • 0 - insert a rest of the currently selected duration
  • Duration is sticky: press a digit once, then keep entering notes/drums at that length until you press another digit.

Curve regions

  • Shift+C works in any mode and always starts/closes a curve region, but does two different things:
  • In Melody mode: starts note-capture. Type notes as usual (A-G) - each one becomes a pitch point instead of its own event. Shift+C again closes the capture and turns the notes into a smooth pitch-bend: one continuous "carrier" event spanning all the notes, plus a sharpness curve that eases between each pitch (the live-keystroke equivalent of a .msh tied group like (DE)).
  • In other modes: starts an empty curve region; Shift+C again just sets its end time. Add points to it afterward with Shift+P (cursor must be inside the region) - a popup lets you choose sharpness or intensity and enter a value.
  • Notes/events entered while the cursor is inside a curve region (but capture is not open) stay fully independent events - a region only modulates whatever overlaps it, it never absorbs notes into curve points on its own.
  • Because Shift+C is claimed globally, C-sharp in Melody mode needs its enharmonic equivalent: Alt+D (D-flat = C-sharp).

Normal mode (press N to enter)

  • T - insert a transient at the cursor
  • C ... C - press once to start a continuous event, move the cursor, press again to set its end
  • A / D / R - popup to set the selected event's attack / decay / release time

Melody mode

  • A-G - insert a note at the cursor, advance cursor by current duration
  • Shift+A-Shift+G - insert a sharp note (except C, see Curve regions above); Alt+A-Alt+G - flat note
  • ! then a note - accent the next inserted note (louder)
  • Octave is chosen automatically for the closest pitch to the last note you typed, the same way MuseScore does it (typing C right after B lands in the next octave up, since that's the nearer C). The very first note of a session uses the manually-set octave below.
  • Ctrl+Up / Ctrl+Down - shift the last-typed note (or the note in progress inside a curve capture) up/down an octave; every note typed after that continues from the new pitch, matching MuseScore's octave-shift behavior

Drums mode

  • K kick, T tom, S snare, H closed hi-hat, X clap, O open hi-hat, C crash, R ride

Editing and clipboard

  • Enter - open the full detail form for the selected item (every field at once)
  • Delete / Backspace - delete the selected item
  • PageUp / PageDown - nudge selected item's intensity; Shift+PageUp/Shift+PageDown nudges sharpness
  • Ctrl+C - copy the selection (or the single selected item if there's no range selection)
  • Ctrl+X - cut the selection (or the single selected item)
  • Ctrl+V - paste at the cursor; a multi-item paste keeps each item's original track

Tracks

  • Tracks are an editor-only way to organize items (e.g. keep transients and continuous events separate so you can range-copy just one kind). They're merged together when you export to .ahap.
  • V - toggle showing only the active track's items in the list
  • Use the Tracks panel below the item list to add, rename (Enter), switch (Up/Down), or delete tracks

Playback

  • P - preview the selected item alone
  • Ctrl+Space - play the whole pattern from the cursor
  • Ctrl+Shift+Space - play the whole pattern from the beginning
  • Escape - stop playback (or clear a selection first, if one is active)

Item details

Send over WebSocket (experimental)

Sends the pattern as JSON to a WebSocket server of your choice, event by event, scheduled by each event's own time. There is currently no known public app that receives this over the air and plays it back - this is here so you can point it at your own receiver (phone app, ESP32 haptic driver, etc) if you build one.
